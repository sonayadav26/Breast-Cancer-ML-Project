# Breast Cancer Prediction using Machine Learning

## Overview
This project uses Machine Learning algorithms to predict breast cancer using the Wisconsin Diagnostic Breast Cancer Dataset.

## Models Used
- Logistic Regression
- SVM
- KNN
- Decision Tree
- Random Forest

## Best Accuracy
98.24%

## Features
- Data preprocessing
- Feature scaling
- ROC Curve
- Confusion Matrix
- Feature Importance
- Correlation Heatmap

## Technologies
Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

## Conclusion
Machine learning algorithms successfully predicted breast cancer with high accuracy.
